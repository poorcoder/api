jQAPI - Alternatif jQuery Dokümasyonu - 1.4.4 sürümü için

Neden?daha hızlı bir dökümantasyon ulaşımı için.

jQuery ana API'si jQuery API Gösterici'si baz alınarak jQuery için Türkçe kaynak oluşturulmak istenmiştir. Ücretsiz olarak indirebilirsiniz.

Klavye KısayollarıHızlandırır!

ESC tuşuna basarak arama kısmına getirebilirsiniz. 

Yukarı yön tuşu      - Önceki kategori / metod
Aşağı yön tuşu    - Sonraki kategori / metod
ENTER tuşu   - Detayları aç kategori / metod
ESC tuşu     - Arama kısmını boşalt ve odakla

Kaynakçave destekçileri.

jQuery, jQuery dokümantasyonu, Highlight Plugin, 'Cowboy' Ben Alman: BBQ Plugin / doTimeout Plugin / geribildirim, yayQuery, Technikwuerze, Erik Zaadi: AIR uygulaması, Dokümantasyonu Türkçeleştiren ekip.
