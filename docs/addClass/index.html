.addClass()

Categories: Attributes | Manipulation > Class Attribute | CSS

.addClass( className ) Returns: jQuery

Açıklama: Eşleşen elementlere her seferinde belirtilen stil(ler) eklenir.

  • Eklendiği versiyon: 1.0.addClass( StilSınıfAdı )

    StilSınıfAdıHer eşleşen elementin class özelliğine bir veya daha fazla stilsınıf ismi ekler.

  • Eklendiği versiyon: 1.4.addClass( function(index, StilSınıfAdı) )

    function(index, class) Bu fonksiyonla bir yada daha fazla birbirinden bir boşlukla ayrılmış StilSınıfı eklenir. Alınan index konumuna göre yeni StilSınıfı eski StilSınıfının üzerine basılır.

Burada dikkat edilmesi gereken husus Bu metod StilSınıfAdını değiştirmez. Basitçe StilSınıfAdı eklenebilecek elementlere StilSınıfAdı ekler.

Eşleşen elementlere tek seferde birden fazla StilSınıfADı eklenmek istendiğinde aralarına bir boşluk konulmalıdır. Aynen şöyle: 

$('p').addClass('myClass yourClass');

Bu metodla birlikte StilSınıfAdını değiştirmek istedindiğinde .removeClass() metodu ile sık sık kullanılır. Aynen şöyle: 

$('p').removeClass('myClass noClass').addClass('yourClass');

Burada myClass ve noClass StilSınıfAdları tüm parağraflardan silindiğinde yourClass StilSınıfAdı eklenmiştir.

jQuery 1.4 de .addClass() metodunun içerisinde fonksiyon calıştırlmasına izin verir. 

$('ul li:last').addClass(function() {
  return 'item-' + $(this).index();
});

Verilen örnekte listedeki beş <li> elementinden son <li> elemente dördüncü elementin StilSınıfAdı eklenmektedir.

Örnekler:

Örnek: Eşleşen elementlere 'selected' StilSınıfAdı eklenmektedir. 

<!DOCTYPE html> <html> <head>
                            <style> p { margin: 8px; font-size:16px; } .selected { color:blue; } .highlight
                            { background:yellow; } </style> <script src="http://code.jquery.com/jquery-latest.min.js"></script>
                            </head> <body> <p>Hello</p> <p>and</p> <p>Goodbye</p>
                            <script>$("p:last").addClass("selected");</script> </body> </html>
